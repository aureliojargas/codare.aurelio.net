Sobre

#!/usr/bin/codare
#
# Feito por programadores para programadores, o CODARE
# é um blog diferente.
#
# Um time de profissionais experientes, que trabalham com
# programação no seu dia a dia, escrevem dicas rápidas e
# práticas sobre os problemas que têm que lidar. Soluções
# funcionais para problemas reais. Sem gordura, sem firula.
#
# Se você é programador, contribua com as dicas colocando
# a sua experiência nos comentários. Conte-nos sobre
# problemas parecidos que enfrentou e qual foi a solução
# utilizada.
#
# Se você é iniciante na arte da programação, leia, aprenda
# e contribua.
#
# Nosso objetivo é reunir a comunidade nacional de
# programadores em torno de um conteúdo de qualidade,
# trocando experiências, sem discriminação de linguagens
# ou plataforma. É sobre código, e codificadores. Participe!
#

CODARE = blog colaborativo de programação

POST = dica rápida, clara e prática, direta ao ponto

LANG = pt_BR

LANGUAGES = [ C, C++, Java, Javascript, PHP, Python, Ruby, Shell, … ]

STYLE = Guia de Estilo para novos posts
